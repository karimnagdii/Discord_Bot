# DJ Sounds Directory

Drop short MP3 clips (ideally 5–10 seconds) into this folder.
The bot will randomly play them when the Unsolicited DJ chaos action fires.

## Suggested clips
- `airhorn.mp3` — Classic airhorn
- `nokia.mp3` — Nokia ringtone (blown out)
- `vine_boom.mp3` — Vine boom sound effect
- `emotional_damage.mp3` — "Emotional damage" meme audio
- `bruh.mp3` — Bruh sound effect

## Sources for royalty-free meme sounds
- https://www.myinstants.com (download button on each sound)
- https://pixabay.com/sound-effects/ (search "meme" or "airhorn")

## Notes
- Only `.mp3` files are played — other formats are ignored.
- Keep clips short (< 15s) so the bot doesn't overstay its welcome in VC.
